The source code and development environment for the OpenIMU335RI can be found through Aceinna's open-source repositories on GitHub and their developer documentation. The platform is designed for use within Visual Studio Code (VS Code) using the PlatformIO IDE extension. 
Source Code Location
Main GitHub Repository: The base library for the OpenIMU335RI is available at: https://github.com/Aceinna/openIMU335RI-lib.
All Aceinna Repositories: You can browse all associated libraries, drivers, and documentation here: https://github.com/orgs/Aceinna/repositories. 
Development Setup
Install VS Code: Download and install Visual Studio Code.
Install PlatformIO Extension: Inside VS Code, install the PlatformIO extension to manage the OpenIMU environment.
Documentation/Guides: Refer to the Aceinna OpenIMU Developer Manual for instructions on setting up the environment: openimu.readthedocs.io. 
Firmware and Tools
Pre-compiled Apps: You can download pre-compiled application firmware via the Aceinna Navigation Studio: https://developers.aceinna.com/code/apps.
Python Driver: A Python-based CLI tool for data logging and device management is available at https://github.com/Aceinna/python-openimu. 
For immediate development, installing the PlatformIO extension and loading the openIMU335RI-lib is the recommended approach to access the full source code.
